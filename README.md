Metro Ticket System
A simple C++ Object-Oriented Programming (OOP) project for a Metro Ticket Booking System. This project demonstrates basic ticket booking, fare calculation, and ticket management using classes and objects.

Features
Station management

Metro route handling

Ticket booking and display

Fare calculation

How to Run
Clone the repository:

text
git clone https://github.com/kit2824bcs003-Abishek/metro-ticket-system.git
Compile the code:

text
g++ -o metro main.cpp
Run the program:

text
./metro
